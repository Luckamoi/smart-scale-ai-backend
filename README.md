# AI-Powered Digital Scale System

## 📌 Overview

This project is an AI-powered digital weighing system that integrates hardware sensors, IoT communication, and backend automation.

This repository focuses on the **software and backend infrastructure** of the system.

## 🧠 System Architecture

The system consists of 3 main components:

### 🔌 Hardware Layer

* Load cell sensor
* Microcontroller (Arduino / ESP32)

### 📡 Communication Layer

* MQTT (Mosquitto) for real-time data transfer

### 🌐 Software & AI Layer (This Repo)

* Flask API for processing data
* PostgreSQL for storage
* n8n for workflow automation
* Cloudflare Tunnel for deployment

## ⚙️ Tech Stack

* Python (Flask)
* Docker / Docker Compose
* PostgreSQL
* n8n
* MQTT (Mosquitto)

## 🚀 Features

* Real-time weight data collection
* IoT-based communication
* Automated workflows
* Scalable backend architecture
